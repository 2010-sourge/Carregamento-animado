# 🔄 Carregamento Animado com HTML e CSS

Projeto desenvolvido em **HTML5** e **CSS3** que apresenta uma animação de carregamento moderna, composta por círculos coloridos em movimento contínuo. É um componente ideal para telas de espera, páginas de carregamento (*loading screen*) e aplicações web.

---

## 📸 Prévia

O projeto exibe um fundo preto com quatro círculos coloridos centralizados, realizando uma animação suave e contínua que proporciona uma experiência visual moderna e dinâmica.

---

## ✨ Funcionalidades

- 🔄 Animação de carregamento contínua
- 🎨 Círculos coloridos com efeito moderno
- ⚡ Movimento suave utilizando CSS
- 🖥️ Layout centralizado na tela
- 📱 Design responsivo
- 💻 Desenvolvido apenas com HTML e CSS

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3

- Transformações CSS
